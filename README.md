# Proyecto Frontend con Angular

## ¿Qué es Angular?
Angular es un framework de desarrollo frontend creado por Google que permite construir aplicaciones web modernas de una sola página (SPA). Utiliza TypeScript y se basa en una arquitectura de componentes para facilitar el desarrollo, mantenimiento y escalabilidad de aplicaciones web.

## ¿Qué herramientas necesitan instalar?
Para desarrollar aplicaciones con Angular es necesario instalar las siguientes herramientas:
- Node.js (entorno de ejecución de JavaScript)
- npm (gestor de paquetes que viene incluido con Node.js)
- Angular CLI (herramienta de línea de comandos para crear y administrar proyectos Angular)
